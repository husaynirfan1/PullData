# Query: what does Status Tab  shows in diagnostic viewer ?

## Table of Contents

- [Sources](#sources)

The **Status Tab** in the Diagnostic Viewer displays **simulator status and related information**, such as:  
1. **Faults**: Indicates if a malfunction is occurring (e.g., blinking icons for failed components).  
2. **Nodes**: Identifies the specific system or component under analysis.  
3. **Severity**: Shows the severity of the fault (e.g., critical, degraded, or minor).  
4. **Time Occurrence**: Displays when the fault was detected.  

Additionally, it provides contextual details like whether the component is powered, loaded, or unloading, and whether it's reporting errors or operating normally. The tab also reflects the system's state, including dependencies and rules that affect computation (e.g., disabled components). 

This information helps users quickly identify issues, assess severity, and track fault propagation across the simulator's hierarchy.
