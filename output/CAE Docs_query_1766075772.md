# Query: how to create label in launchpad ?

## Table of Contents

- [Sources](#sources)

To create a label in LaunchPad, follow these steps based on the provided context:

1. **Manual Label Creation**:
   - Open the **Configuration Management (CM) perspective**.
   - Right-click on a package or entity and select **Advanced > Create Label**.
   - Enter a custom label name and confirm. Existing labels will be displayed for reference.

2. **Auto Label Creation**:
   - Right-click on a package or entity and select **Advanced > Create Auto Label**.
   - The label name defaults to the current date and time (e.g., `label_20231015-1430`).

3. **Bulk Labeling**:
   - Select multiple entities (e.g., all packages) and use **Advanced > Create Label** to apply the label to all selected items.

4. **Using the CM Action Menu**:
   - The process is similar to the above, but you can also access it via the **CM action menu** in the UI.

5. **Optional: Labeling Packages**:
   - To label a package, use **Select Package Version** from the CM menu and apply the label manually.

**Note**: Auto-labels are handled automatically by LaunchPad, while manual labels require user input. Ensure you have the correct permissions and configurations (e.g., LDAP sync) to proceed.
