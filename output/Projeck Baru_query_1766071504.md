# Query: How to do create label on launchpad?

## Table of Contents

- [Sources](#sources)

To create a label in LaunchPad, follow these steps based on the provided context:

1. **Access the Configuration Management Menu**:  
   - Right-click on the package or entity you want to label.  
   - Select **Advanced > Create Label** from the dropdown menu.

2. **Create a Custom Label**:  
   - Enter a name for the new label in the dialog box.  
   - Confirm your selection to create the label manually.

3. **Auto Label (Automatic Creation)**:  
   - Right-click the package and select **Advanced > Create Auto Label**.  
   - The label name will be automatically generated using the current date and time (default).  

4. **Select an Older Label**:  
   - If you want to revert to an existing label for a package, right-click the package and choose:  
     - **File > Advanced > Export LDAP Users into StarTeam** (for user synchronization) or  
     - **Select Package Version** to manage labels via the Configuration Management menu.  

This process allows users to customize labels manually or let LaunchPad handle automatic label creation.
